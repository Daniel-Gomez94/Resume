Project One

Contact Manager Website

Description

This project was a full-stack web application for managing personal contacts. It used a LAMP stack (Linux, Apache, MySQL, PHP) deployed on DigitalOcean. The app included user authentication, CRUD functionality for contacts, and integration with SwaggerHub for documenting the API endpoints.

Skills Learned

  • MySQL database design and CRUD queries
  • PHP for backend API endpoints
  • HTML, CSS, and JavaScript for frontend
  • Deployment on DigitalOcean with domain setup
  • API documentation with SwaggerHub
